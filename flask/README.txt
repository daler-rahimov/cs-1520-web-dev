To  activate the virtual env run : source pythonVM/bin/activate 
Or activate already created env run: . [envFolder]/bin/ectivate
To deactivate: deactivate
To start a new virual python machine: 
	virtualenv [dir]
	pip install Flask  
to run a flask app:
	export FLASK_APP=hello.py
	flask run --host=0.0.0.0
